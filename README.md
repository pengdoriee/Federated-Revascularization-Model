## Federated-Revascularization-Model
- This repository contains the implementation of a **Federated learning approach to optimize coronary revascularization decisions with privacy-preserving data integration from multiple hospitals**
<br>
<br>
<br>

## Federated Learning Platform
- We use ELF platform as the federated aggreagation server for FedAVG.
- https://github.com/CNTBMS/ELF
<br>
<br>
<br>

## What's Inside
- feature selection: Anova F-value, Chi-square test, and Mutual information
- standardization
- imputation: iterative imputation with chained equations
- inital global model: The architecture of initial global model
- requirements.txt
